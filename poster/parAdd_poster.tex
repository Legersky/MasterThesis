\block{Abstract}{
Parallel addition is used for instance as a part of division algorithm  in computer power units with base 4 and an integer alphabet.  An extending window method, which is due to M. Svobodov\'a, constructs parallel addition algorithms in more general numeration systems. 

We discuss convergence of the method and we develop tools to reveal possible non-convergence. 

We implemented the method with several inner modifications in SageMath. The easy control of non-convergence allowed to test more than 5000 different numeration systems to obtain a parallel addition algorithm in about 80 cases.
}
